Instrucciones para Descargar y Ejecutar

  1. Clonar los repositorios desde GitHub usando el comando git clone.
  2. Las siguientes instrucciones funcionan tanto para el proyecto de MongoDB como para el de Firebase
    • Navegar al directorio del proyecto.
    • Ejecutar el comando npm install para instalar las dependencias.
    • Iniciar el proyecto con node app.js.
  3. Para acceder a la Documentación del proyecto, puedes hacer clic en el enlace que arroja la terminal cuando ejecutes el proyecto.
